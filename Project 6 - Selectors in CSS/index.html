Selectors used in CSS:

  • Simple Selectors
  • Combinator Selectors
  • Attribute Selectors
  • Pseudo-Class Selectors
  • Pseudo-Element Selectors

Simple Selectors

  • Element Selector: Selects all occurrences of a specific HTML element.
  • Class Selector: Selects all elements with the class btn.
  • ID Selector: #header Selects the element with the ID header.
  • Universal Selector: *{} Selects all elements on the page.

Combinator Selectors

  • Descendant Selector: A space B Selects all B elements that are inside A, at any nesting level.
  • Child Selector: A>B Selects only B elements that are direct children of A.
  • Adjacent Sibling Selector: A+B Selects the first B element immediately after A.
  • General Sibling Selector: A~B Selects all B elements that are siblings of A and come after it.

Attribute Selectors

  • Has Attribute [attr]: Selects elements that have the specified attribute.
  • Exact Match [attr="value"]: Selects elements whose attribute value exactly matches the specified value.
  • Start with ^= Selects elements whose attribute value starts with the specified value.
  • Ends with $= Selects elements whose attribute value ends with the specified value.
  • Contains *=< Selects elements whose attribute value contains the specified value./li>
  • Space-Separated Value ~= Selects elements whose attribute contains the specified word in a space-separated list.
  • Hyphen-Separated Value |= Selects elements with lang="en" or values starting with en-.

Pseudo-Class Selectors

    These are most commonly used:

  • :hover Selects an element when the mouse pointer is over it.
  • :active Selects an element while it is being clicked
  • :focus Selects an element that currently has focus.
  • :visited Selects links that have already been visited.
  • :link Selects links that have not been visited yet.
  • :first-child Selects an element that is the first child of its parent.
  • :nth-child(n) Selects an element based on its position among siblings.
  • :last-child Selects an element that is the last child of its parent.
  • :checked Selects checked radio buttons and checkboxes.
  • :disabled Selects disabled form elements.
  • :first-of-type Selects the first occurrence of a specific element type.
  • :nth-of-type(n) Selects an element of a specific type based on its position.
  • :last-of-type Selects the last occurrence of a specific element type.
  • :only-child Selects an element that is the only child of its parent.
  • :invalid Selects form fields with invalid input.
  • :not Selects elements that do not match the specified selector.
  • :root Selects the root element ().
  • :target Selects an element whose ID matches the URL fragment.
  • :empty Selects elements with no child elements or text content.
  • :required Selects form fields that are required.
  • :valid Selects form fields with valid input.

Pseudo-Element Selectors

  • ::before Inserts content before an element's content.
  • ::after Inserts content after an element's content.
  • ::first-letter Selects the first letter of a text block.
  • ::first-line Selects the first line of a text block.
  • ::placeholder Styles the placeholder text of form controls.
  • ::selection Styles text selected by the user.
  • ::marker Styles list item markers (bullets/numbers).
  • ::file-selector-button Styles the button of a file input.
  • ::backdrop Styles the background layer behind fullscreen elements or dialogs.
  • ::cue Styles captions/subtitles in media elements.
  • ::part() Selects and styles exposed parts of a Web Component.
  • ::slotted() Selects elements passed into a Web Component slot.
  • ::target-text Styles text targeted/highlighted by a URL fragment.
