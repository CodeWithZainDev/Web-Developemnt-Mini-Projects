ADD AUDIO

Atrributes used in Audio:

  • controls: Displaying Controls on Audio
  • loop: Repeating in a loop until you stop
  • muted: sound muted when the page load, then you can unmute manually
  • autoplay: Audio in autoplay mode after page load
  • src: Defines the location file audio you wanna to add for users

  • preload: Contains three further attributes;

  • none: Don't preload media which means audio doesn't download when page load or refresh later
  • metadata: It determines the information of an audio to browsers and shows to user like display length of audio before play where user can see the full length of audio before begins.
  • auto: When this attribute used, audio automatically downloading in browser and requires user's data to download that audio and user can play that audio and listen without interuptions but using of this attribute may result downgrade speed at loading of a webpage
